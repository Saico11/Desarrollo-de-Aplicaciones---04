¿Qué son las Responsive Web Design?

Responsive web design o diseño web adaptable es la capacidad de que una página se adapte al tamaño de distintos dispositivos a comparación de las páginas tradicionales que estaban diseñadas únicamente para un tamaño o formato específico. Para esto se trabaja con los Media Queries que permiten establecer reglas que indicarán cuando se ejecutará una configuración CSS dependiendo del tamaño de la página. (w3schools, s.f.)

Estrategias de las Responsive Web Design

Mobile First

Con esta estrategia se definen las configuraciones CSS que se aplicarán primeramente para dispositivos móviles, a continuación las que se aplicarán para tabletas y finalmente las que se aplicarían para computadoras de escritorio o laptops.

Desktop First

Con esta estrategia se definen las configuraciones CSS en primer lugar para computadoras de escritorio o laptops, a continuación las que se aplicarían para las tabletas y finalmente las que se aplicarían para dispositivos móviles.
